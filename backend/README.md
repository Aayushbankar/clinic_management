## Backend (PHP)

This folder contains a small PHP JSON API used by the CMS frontend.

### Entry points
- `public/api.php`: JSON API router (recommended)
- `public/index.php`: simple health page

### Environment variables (recommended)
- `CMS_DB_HOST` (default: `127.0.0.1`)
- `CMS_DB_PORT` (default: `3306`)
- `CMS_DB_NAME` (default: `clinic_management`)
- `CMS_DB_USER` (default: `root`)
- `CMS_DB_PASS` (default: empty)
